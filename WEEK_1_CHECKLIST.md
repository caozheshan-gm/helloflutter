# Flutter 学习清单（顺序版）

## 第 1 轮基础清单

- [ ] 运行 `flutter doctor`，修复所有会阻碍 iOS 开发的报错或警告。
- [ ] 在 `hello_flutter` 目录启动项目，确认 iOS 模拟器可正常运行。
- [ ] 阅读并解释 `main.dart` 中 `main -> runApp -> MaterialApp` 的调用链。
- [ ] 新建一个 `StatelessWidget` 卡片组件并在首页展示。
- [ ] 将首页计数器区域拆分成独立组件，减少 `main.dart` 体积。
- [ ] 使用 `Row`、`Column`、`Container`、`Padding` 重新组织首页布局。
- [ ] 提取统一间距与主题色常量，避免页面里硬编码样式值。
- [ ] 实现“加一 / 减一 / 重置”三个按钮并接入 `setState`。
- [ ] 增加边界处理（例如计数不小于 0）。
- [ ] 新建一个 Todo 页面，支持输入并新增条目。
- [ ] 为 Todo 页面实现删除功能与空状态提示。
- [ ] 完成首页到 Todo 页的跳转与返回（`push/pop`）。
- [ ] 在页面跳转时传递一个简单参数并在目标页展示。
- [ ] 编写一份简短复盘：做得好、卡住点、下一步优化项。
